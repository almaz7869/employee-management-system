# Employee Management System (EMS)

A Full Stack Employee Management System developed using **React.js**, **Spring Boot**, and **MySQL**.

This project helps administrators manage employee records efficiently with advanced dashboard analytics, employee reports, PDF/Excel export, and salary visualization.

---

## 🚀 Features

- Admin Login Authentication
- Add New Employee
- Edit Employee Details
- Delete Employee
- Employee Photo Support
- Search Employee by Name/Department
- Total Employee Counter
- Total Salary Counter
- Salary Bar Chart
- Department Pie Chart
- Reports Dashboard
- Download PDF Report
- Download Excel Report
- Responsive Professional UI
- Dark Mode Toggle

---

## ⚙️ How to Run Project

### Backend

1. Open Spring Boot project in STS / IntelliJ
2. Configure MySQL database in `application.properties`
3. Run `EmsbackendApplication.java`

Backend runs on:

http://localhost:8080

---

### Frontend

1. Open terminal inside `ems-frontend`
2. Run:

npm install  
npm start

Frontend runs on:

http://localhost:3000
